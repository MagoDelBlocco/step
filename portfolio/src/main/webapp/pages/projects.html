Personal Projects

This page contains all the projects that I was involved in:

Designed and implemented a C++ program built from the ground up that uses machine learning to recognize digits from png images. Trained using the MNIST dataset.

Built a C++ program that simulates how a ride-service hailing app handles queries from customers. It takes into consideration how much fuel a driver has, it's average speed and the amount of time the client is willing to wait

Developed a C++ program which has a basic graphical interface, can allow the user to play the Flappy Bird game, or can let an ANN train to play it, using an evolutionary algorithm.

Implemented a Matlab program that is able to be trained on portraits of people, and create an eigenspace of human faces. Based on that, it can recognize if the image presented is a portrait of a human.

Created a C++ chess engine, using the Chess Engine Communication Protocol. It is able to reach a depth of 10 while searching in the minimax tree, using heuristics like MTD(f), delta pruning, killer move heuristic.
